The Lost & Found Mobile App is a final year project that improves how lost property is managed on a college campus. It provides a central place where users can report lost items, upload found items, browse listings, and support faster recovery of personal belongings.

### What the project does
- Allows users to report lost items
- Allows users to upload found items
- Makes it easier to browse and recover belongings on campus
